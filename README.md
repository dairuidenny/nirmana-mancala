# Nirmana Mancala

这是 Coincidence 团队制作的编程解谜游戏 **《U.V.S. Nirmana》** 内的一个播棋小游戏。
因为笔者玩得很上头，所以把它做（vibe coding）成了一个独立的网页版本，便于在任意平台随时游玩。
原游戏的 steam 页面链接：https://store.steampowered.com/app/2536720/UVS/

### 核心规则
1. **棋盘**：12 个环形分布的槽位，初始随机分布 50 枚棋子。
2. **播种**：选择一个槽位取出所有棋子，沿**顺时针**方向每个槽位放入 1 枚。
3. **基础得分**：最终落点槽位若正好有 2 或 3 枚棋子，则将其移除并得分，分数等于移除的棋子数。
4. **平方连锁**：得分后，**逆时针**检查相邻槽位。若也满足 2 或 3，则继续移除并获得数量 **平方** 的额外分数。可连锁移除直到不满足 2 或 3。
5. **生存压力**：若某回合未得分，进入“危险状态”。下一回合必须得分，否则直接游戏失败。
6. **终局**：清空棋盘上所有棋子后游戏结束，追求最高总分。

### 技术说明
- HTML5 Canvas + JavaScript 实现，无外部依赖。
- 响应式设计，适配桌面及移动端基础体验。
